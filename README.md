<h2 align="left">Sobre el projecte</h2>

###

<p align="left">El Joc d’en Pingu és un joc de taula digital desenvolupat en Java amb JavaFX, inspirat en el clàssic Escales i Tobogans. Els jugadors controlaran pingüins que han d’avançar per un taulell amb diverses caselles i events. Hauran de superar obstacles, gestionar el seu inventari i enfrontar-se a altres jugadors o inclús a una CPU amb intel·ligència artificial.<br><br>L'objectiu del projecte és aplicar coneixements de desenvolupament en Java, bases de dades, anàlisi i disseny d’interfícies i diagrames per crear un joc funcional i dinàmic. A més, es fomentarà la presa de decisions estratègiques i la planificació del desenvolupament mitjançant un Pla de Digitalització, que cada equip haurà de presentar.</p>
